### README - zik-et-accords-v1

# zik-et-accords-v1

**zik-et-accords-v1** est une application web simple permettant d'afficher des sons, des paroles (lyrics) et des accords. Dans cette version, les sons sont statiques et doivent être ajoutés directement dans le code source.

## Fonctionnalités
- Affichage des sons avec leurs paroles et accords.
- Les données des sons et accords sont définies dans un fichier JavaScript statique.
- Ajout manuel de nouveaux sons par le développeur via modification directe du code.

## Technologies utilisées
- **HTML5**
- **CSS3**
- **JavaScript**
- **Bootstrap 4**

## Installation et utilisation
1. Cloner le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/zik-et-accords-v1.git
   ```

2. Ouvrir le fichier `index.html` dans un navigateur pour visualiser l'application.

3. Pour ajouter un nouveau son :
   - Les nouveaux sons doivent être ajoutés manuellement par le développeur dans le fichier JavaScript correspondant.
   - Pour une version avec publication dynamique par les utilisateurs, veuillez consulter la v2 de l'application :
     ```bash
     https://github.com/ELDBBAL/zik-et-accords-v2
     ```

## Structure du projet
- `index.html` : Page principale affichant les sons.
- `js/` : Dossier contenant les fichiers JavaScript, y compris les données des sons.
- `styles/` : Dossier contenant les fichiers CSS.
- `images/` et `plugins/` : Dossiers pour les ressources additionnelles.
